Descrição do Sistema

Esse sistema é voltado principalmente para a utilização de clínicas protéticas, auxiliando na gestão de trabalhos e pacientes. Ele foi desenvolvido com a fácil modulação do sistema, permitindo adaptações específicas para cada empresa.

É um sistema de fácil utilização, pensado para melhorar a eficiência dos processos nas clínicas.

Ao acessar o sistema, iniciara com uma tela de login

Caso ja tenha cadastro, sera necessário apenas fazer login com seu e-mail e senha

Caso seja um novo usario terá a opção de Cadastre-se aqui


Para entrar no sistema, no cabeçalho, possui a palavra acesse o sistema, que ira te direcionar para a tela de login
